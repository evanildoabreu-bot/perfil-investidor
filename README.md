# Perfil de Investidor

Questionário interativo para análise de perfil de investidor em português.

## 🎯 Sobre

Uma aplicação web em HTML/CSS/JavaScript que avalia o perfil de investidor do usuário através de 10 perguntas, classificando em:
- Conservador
- Moderado
- Moderado Arrojado
- Arrojado

## 🚀 Como usar

1. Abra o arquivo `index.html` no navegador
2. Responda as 10 perguntas
3. Receba sua análise de perfil com:
   - Classificação de risco
   - Tolerância à volatilidade
   - Alocação sugerida
   - Produtos compatíveis por fase

## 📋 Perguntas cobertas

1. Objetivo principal de investimento
2. Horizonte de tempo
3. Reação a quedas de 15% na carteira
4. Experiência com investimentos
5. Percentual disposto a alocar em ativos de risco
6. Situação de renda e reserva de emergência
7. Preferência de tipo de retorno
8. Conhecimento de produtos de investimento
9. Necessidade de liquidez
10. Maior preocupação ao investir
